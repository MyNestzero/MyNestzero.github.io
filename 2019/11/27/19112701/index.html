MAC中Workbench的使用一

新建MySQL的数据库实例
  1. 打开workbench，在【Migration】页面中的【MySQL Connections】中选择一个你要建立数据库实例的地方，如图中给的【Local instance 3306】
  2. 如图所示，右击，在弹出的卡片中选择【Open connection】
  3. 在下个界面中找到【Schemas】在其空白面板上右击，选择【create schema…】,如图所示
  4. 在弹出的界面【new_schema-Schema】中，输入要建立数据库实例的名字，编码格式，然后点击【apply】
  5. 在弹出的界面继续点击【apply】，数据库实例建立完成
