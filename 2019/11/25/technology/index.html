MAC安装mysql

  1. 查看可以安装的mysql版本：
    brew search mysql
    结果如下：
  2. 安装mysql：
    brew install mysql@5.7
    出现以下显示表示安装成功：
  3. 配置环境变量：在bash_profile中添加如下：
  4. 输入如下命令启动mysql服务
    mysql.server start
    结果如下：
  5. 进入mysql命令行命令
    mysql -uroot //这是默认情况下的账号,没有设置用户名密码的root账户
    mysql -u root -p //使用用户名密码登录mysql
    mysql.server stop // 停止mysql服务
  6. 修改mysql的用户名以及密码
    mysql_secure_installation //可以设置初始用户的密码
  7. 登录mysql
